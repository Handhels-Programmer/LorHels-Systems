LorHels ERP

Es un potente ERP SaaS en la nube. Gestiona ventas, compras, inventario y finanzas en tiempo real. Destaca por facturación con PDFs modernos, cobros por WhatsApp, control de deudas (CXC/CXP), comisiones automáticas y un panel administrador para gestionar suscripciones de empresas.

HTML5 JavaScript Tailwind CSS Lucide Icons Firebase Firebase Authentication Cloud Firestore Chart.js html2pdf.js

LorHels CV

Es un moderno SaaS de reclutamiento. Candidatos crean su CV con QR y empresas lo escanean. Destaca por su exportación a PDF, modelo Freemium validado por WhatsApp y un panel Admin para controlar perfiles y pagos. ¡Tu app está lista para brillar.

HTML5 Tailwind CSS JavaScript Firebase SDK Firebase Authentication Firebase Firestore QRCode.js html2pdf.js FontAwesome

LorHels Music

Es un clon de Spotify profesional. Tiene roles (Admin/Premium/Básico), pagos vía WhatsApp y reproductor fluido. Destaca su radio 24/7 sincronizada, modo offline para descargar MP3, playlists compartibles y panel admin total.

HTML5 CSS3 JavaScript Tailwind CSS Lucide Icons Firebase Authentication Firebase Firestore HTML5 Audio API Cache API

LorHels Barber

App web completa. Clientes tienen reservas inteligentes, disponibilidad en vivo y catálogo dinámico. Admin controla Punto de Venta (POS), inventario, caja diaria y staff. Hoy corre como Demo Offline local.

HTML5 Tailwind CSS JavaScript Web Storage API Lucide Icons Google Fonts Firebase

LorHels Domino

App de dominó en tiempo real. El Admin gestiona mesas, edita nombres y registra deudas/pagos. El Invitado solo observa deudas y un contador de pérdidas por jugador. Incluye historial de caja, cierre de partidas seguro y sincronización instantánea con Firebase.

Firebase Firestore Firebase Auth Tailwind CSS Glassmorphism JavaScript Lucide Icons

LorHels Viewer

Es una app web B2B para estudios de diseño. Centraliza modelos 3D interactivos, galerías comparativas y gestión de clientes. Destaca por su sistema de tickets en tiempo real para organizar el feedback bajo un panel seguro.

HTML5 JavaScript Tailwind CSS Phosphor Icons Three.js Iframes Firebase Authentication Cloud Firestore Cloudinary
